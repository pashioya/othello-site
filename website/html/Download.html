How to use Integrity Check?

  • Download the userIntegrityCheck
  • Unzip userIntegrityCheck.zip
  • Place the game file/folder you want to check in the userIntegrityCheck folder
  • Open (Windows)cmd or Terminal(Mac/Linux)
  • if on Windows follow the instructions at the bottom
  • Enter the command "cd Download/userIntegrityCheck/" (If your files are in a different directory replace with the appropriate directory)
  • Enter the command "bash userIntegrityScript.sh"
  • The output should tell you whether your file is up to date
